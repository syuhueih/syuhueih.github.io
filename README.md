# Dynamic Portfolio Site

This is a standalone dynamic version of Syu-Huei Huang's personal website.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `app.js`: content data and interactions
- `assets/images/`: copied images and logos
- `assets/videos/`: project demo videos
- `assets/files/`: resume PDF

## Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8082 --directory /Users/syuhueihuang/Documents/sh-agent/outputs/dynamic-portfolio-site
```

Then visit:

```text
http://localhost:8082/
```

## Edit Content

Most visible content is in `app.js`:

- `experiences`
- `projects`
- `interests`
- `news`
- `publications`
